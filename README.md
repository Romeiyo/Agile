# Conference Room Booking System

## About
The **Conference Room Booking System** is designed to streamline the management of office meeting spaces. It enables employees to secure workspaces, assists receptionists in managing visitor bookings, and provides facilities managers with tools to schedule maintenance.  

**Current Sprint Goal:** Enable core booking functionality for Employees, Receptionists, and Facility Managers.

## Key Features
The system is built around two main Epics: **Core Booking Foundation** and **Administrative Control**.

### Currently Implemented (Sprint 1)
* **Basic Room Booking:** Employees can view availability and book conference rooms for specific dates and times.
* **Visitor Booking Assistance:** Receptionists can view available rooms and book on behalf of external guests.
* **Conflict Prevention:** The system automatically detects and prevents double bookings or invalid dates.

### Upcoming Features (Roadmap)
* **Admin Dashboard:** A centralized view for monitoring room utilization and activity.
* **Maintenance Scheduling:** Tools for Facilities Managers to block off rooms for repairs.
* **Recurring Meetings:** Functionality for employees to set repeat bookings.
* **Room Filtering:** Ability to filter rooms by capacity and equipment.
